Proxy settings (required for CORS)

YouTube's API sends no CORS headers, so the browser can't call it directly — every request goes through a proxy. On Vercel this is auto-detected as /api/proxy?url=. On GitHub Pages, deploy the included Vercel function (or any CORS proxy) and paste its base here. Use {url} as a placeholder, or end with ?url= to have the target appended (URL-encoded).
